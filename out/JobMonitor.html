getJobs()

Get the list of currently active jobs from qstat, and emit event according to the completion status of each registered job: if a job is registered but does not show up on the list returned by qstat, it emits a "JobCompleted" event with the id of the job as a message; otherwise, if it appears on the list but its status is marked as "ERROR", emit the event "JobError" along with the id of the job.
